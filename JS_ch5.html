Map

Recall, that map is an operation on an array. It is designed to take a function that takes one parameter - the current item being iterated on in the map.

So a look at the example :

var my_array = [1,2,3];

Let's build a function timesTwo that takes an value and returns it multiplied by 2 :

var timesTwo = function (a) {
    return a * 2;
}

One way to do this is :

var my_array_timesTwo = [];
for (var i = 0 ; i  < my_array.length; i++ ) {
    my_array_timesTwo.push(my_array[i]);
}

Works, but is a little too much code. Another way is to use our built in forEach operator on arrays, which takes a function with a single parameter (the item currently iterated on in the array) :

var my_array_timesTwo = [];
my_array.forEach( function(d) { my_array_timesTwo.push (d) } );

Yet another way is to use map. Remember, map takes a function as input and returns a list of the array items having had that function run on each item in it :

var my_array_timesTwo = [];
my_array_timesTwo = my_array.map( function (d) { return timesTwo(d); } );

Convince yourself this works.
